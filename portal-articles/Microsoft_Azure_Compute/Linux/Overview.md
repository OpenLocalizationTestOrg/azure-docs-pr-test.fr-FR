# <a name="linux-virtual-machines-documentation"></a>Documentation sur les machines virtuelles Linux

Les Machines virtuelles Linux Azure fournissent une infrastructure sécurisée et virtualisée à la demande et à grande échelle avec Red Hat, Ubuntu ou la distribution Linux de votre choix. Découvrez comment créer, configurer, gérer et mettre à l’échelle des machines virtuelles Linux avec nos guides de démarrage rapide, nos didacticiels et nos exemples.
