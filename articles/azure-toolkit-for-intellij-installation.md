<span data-ttu-id="3f512-104">Le kit de ressources Azure pour IntelliJ contient des modèles et des fonctionnalités qui vous permettent de créer, de développer, de tester et de déployer des applications Azure avec l’environnement de développement IntelliJ IDEA.</span><span class="sxs-lookup"><span data-stu-id="3f512-104">The Azure Toolkit for IntelliJ provides templates and functionality that allow you to easily create, develop, test, and deploy Azure applications using the IntelliJ IDEA development environment.</span></span> <span data-ttu-id="3f512-105">Le kit de ressources Azure pour IntelliJ est un projet Open Source, dont le code source est disponible sous licence MIT sur le site du projet sur GitHub à l’adresse suivante :</span><span class="sxs-lookup"><span data-stu-id="3f512-105">The Azure Toolkit for IntelliJ is an Open Source project, whose source code is available under the MIT License from the project's site on GitHub at the following URL:</span></span>

<span data-ttu-id="3f512-106"><https://github.com/microsoft/azure-tools-for-java></span><span class="sxs-lookup"><span data-stu-id="3f512-106"><https://github.com/microsoft/azure-tools-for-java></span></span>

<span data-ttu-id="3f512-107">Il existe deux méthodes d’installation du kit de ressources Azure pour IntelliJ : à partir de la boîte de dialogue Settings (Paramètres) et à partir du menu Configure (Configurer) de l’écran d’accueil ; les deux méthodes d’installation sont présentées dans les étapes suivantes.</span><span class="sxs-lookup"><span data-stu-id="3f512-107">There are two methods of installing the Azure Toolkit for IntelliJ, from the Settings dialog box and from the Configure menu on the start screen; both installation methods will be demonstrated in the following steps.</span></span>

## <a name="to-install-the-azure-toolkit-for-intellij-from-the-settings-dialog-box"></a><span data-ttu-id="3f512-108">Pour installer le Kit de ressources Azure pour IntelliJ à partir de la boîte de dialogue Settings (Paramètres)</span><span class="sxs-lookup"><span data-stu-id="3f512-108">To install the Azure Toolkit for IntelliJ from the settings dialog box</span></span>
1. <span data-ttu-id="3f512-109">Démarrez IntelliJ IDEA.</span><span class="sxs-lookup"><span data-stu-id="3f512-109">Start IntelliJ IDEA.</span></span>
2. <span data-ttu-id="3f512-110">Une fois IntelliJ IDEA ouvert, cliquez sur **File** (Fichier), puis sur **Settings** (Paramètres).</span><span class="sxs-lookup"><span data-stu-id="3f512-110">When the IntelliJ IDEA opens, click **File**, then click **Settings**.</span></span>
   
    ![Ouvrir la boîte de dialogue Settings (Paramètres) d’IntelliJ IDEA][01a]
3. <span data-ttu-id="3f512-112">Dans la boîte de dialogue Settings (Paramètres), cliquez sur **Plugins** (Plug-ins), puis sur **Browse repositories** (Parcourir les référentiels).</span><span class="sxs-lookup"><span data-stu-id="3f512-112">In the Settings dialog box, click **Plugins**, and then click **Browse repositories**.</span></span>
   
    ![Boîte de dialogue Settings (Paramètres) d’IntelliJ IDEA][02a]
4. <span data-ttu-id="3f512-114">Dans la boîte de dialogue **Browse Repositories** (Parcourir les référentiels) , tapez « Azure » dans la zone de recherche.</span><span class="sxs-lookup"><span data-stu-id="3f512-114">In the **Browse Repositories** dialog box, type "Azure" in the search box.</span></span> <span data-ttu-id="3f512-115">Mettez en surbrillance **Azure Toolkit for IntelliJ** (Kit de ressources Azure pour IntelliJ), puis cliquez sur **Install** (Installer).</span><span class="sxs-lookup"><span data-stu-id="3f512-115">Highlight **Azure Toolkit for IntelliJ**, and then click **Install**.</span></span>
   
    ![Rechercher le kit de ressources Azure pour IntelliJ][03]
   
    <span data-ttu-id="3f512-117">IntelliJ IDEA affiche la progression de l’installation dans une boîte de dialogue.</span><span class="sxs-lookup"><span data-stu-id="3f512-117">IntelliJ IDEA will display the installation progress in a dialog box.</span></span>
   
    ![Progression de l’installation][04]
5. <span data-ttu-id="3f512-119">Une fois l’installation terminée, cliquez sur **Restart IntelliJ IDEA**(Redémarrer IntelliJ IDEA).</span><span class="sxs-lookup"><span data-stu-id="3f512-119">When the installation has completed, click **Restart IntelliJ IDEA**.</span></span>
   
    ![Restart IntelliJ IDEA][05]
6. <span data-ttu-id="3f512-121">Cliquez sur **OK** pour fermer la boîte de dialogue Settings (Paramètres).</span><span class="sxs-lookup"><span data-stu-id="3f512-121">Click **OK** to close the Settings dialog box.</span></span>
   
    ![Fermer la boîte de dialogue Settings (Paramètres) d’IntelliJ IDEA][06]
7. <span data-ttu-id="3f512-123">Quand vous êtes invité à redémarrer IntelliJ IDEA ou à reporter l’opération, cliquez sur **Restart**(Redémarrer).</span><span class="sxs-lookup"><span data-stu-id="3f512-123">When prompted to restart IntelliJ IDEA or postpone, click **Restart**.</span></span>
   
    ![Restart IntelliJ IDEA][07]
