# <a name="irregular-sign-in-activity"></a>Activité de connexion anormale
Connexions anormales sont celles qui ont été identifiées par nos algorithmes, sur la base de hello d’une condition « voyage impossible » associée à un emplacement et le périphérique de connexion anormale d’apprentissage. Cela peut signifier qu'un pirate est parvenu à se connecter à l'aide de ce compte.
Nous vous enverrons un toohello notification par courrier électronique pour les administrateurs généraux si nous rencontrons anormales ou 10-événements de connexion dans un intervalle de 30 jours ou moins. Soyez sûr de tooinclude aad-alerts-noreply@mail.windowsazure.com dans votre liste d’expéditeurs fiables.
