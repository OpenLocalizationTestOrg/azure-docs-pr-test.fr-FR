## <a name="data-retention"></a><span data-ttu-id="eca3c-132">Conservation des données</span><span class="sxs-lookup"><span data-stu-id="eca3c-132">Data retention</span></span>

<span data-ttu-id="eca3c-139">**Suppression de données obsolètes**: les données qui n’ont pas été utilisées depuis un an (« la période de rétention ») sont considérées comme obsolètes et peuvent être supprimées d’Azure.</span><span class="sxs-lookup"><span data-stu-id="eca3c-139">**Stale data deletion**: Data that has not been accessed for one year (“the retention period”) will be treated as stale and may be deleted from Azure.</span></span> <span data-ttu-id="eca3c-140">La période de rétention est susceptible de changer, mais ne sera pas inférieure à 90 jours.</span><span class="sxs-lookup"><span data-stu-id="eca3c-140">The retention period is subject to change but will not be less than 90 days.</span></span> <span data-ttu-id="eca3c-141">Il peut s’agir d’un ensemble spécifique de paramètres Windows/d’application ou de tous les paramètres d’un utilisateur.</span><span class="sxs-lookup"><span data-stu-id="eca3c-141">The stale data may be a specific set of Windows/application settings or all settings for a user.</span></span> <span data-ttu-id="eca3c-142">Par exemple :</span><span class="sxs-lookup"><span data-stu-id="eca3c-142">For example:</span></span>
