## <a name="suitable-scenarios-for-the-oauth2-implicit-grant"></a><span data-ttu-id="927a6-121">Scénarios adaptés à l’octroi implicite OAuth2</span><span class="sxs-lookup"><span data-stu-id="927a6-121">Suitable scenarios for the OAuth2 implicit grant</span></span>
<span data-ttu-id="927a6-122">Comme la spécification OAuth2 l’indique, l’octroi implicite a été conçu pour autoriser les applications d’agent utilisateur, c’est-à-dire les applications JavaScript
