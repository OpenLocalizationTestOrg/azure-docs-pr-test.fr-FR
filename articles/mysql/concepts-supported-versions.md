# <a name="supported-azure-database-for-mysql-server-versions"></a>Versions prises en charge du serveur de base de données Azure pour MySQL
Base de données Azure pour MySQL a été développé à partir de MySQL Community Edition [MySQL Community Edition](https://www.mysql.com/products/community/), à l’aide du moteur de InnoDB hello.  Base de données Azure pour MySQL prend actuellement en charge les versions suivantes de hello.

## <a name="managing-updates-and-upgrades"></a>Gestion des mises à jour et des mises à niveau
La base de données Azure pour MySQL gère automatiquement les correctifs pour les mises à jour de version mineures. En préversion publique, les mises à niveau de version majeures ne sont pas possibles pour le moment. Par exemple, la mise à niveau à partir de MySQL 5.6 tooMySQL 5.7 n’est pas pris en charge.
