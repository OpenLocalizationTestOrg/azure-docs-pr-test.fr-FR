# <a name="manage-virtual-machine-extensions"></a>Gestion des extensions de machine virtuelle
Décrit comment toofind, ajouter, modifier ou supprimer des Extensions de machine virtuelle Windows ou des ordinateurs virtuels Linux sur Azure.

> [!IMPORTANT] 
> Azure dispose de deux modèles de déploiement différents pour créer et utiliser des ressources : [le déploiement Resource Manager et le déploiement classique](../../../resource-manager-deployment-model.md). Cet article décrit à l’aide du modèle de déploiement classique hello. Microsoft recommande que la plupart des nouveaux déploiements de modèle du Gestionnaire de ressources hello. Pour plus d’informations sur les extensions de machine virtuelle à l’aide du modèle de gestionnaire de ressources hello, consultez [ici](../extensions-features.md?toc=%2fazure%2fvirtual-machines%2fwindows%2ftoc.json).
