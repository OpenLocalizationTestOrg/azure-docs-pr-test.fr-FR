# <a name="deploy-popular-application-frameworks-on-linux-using-azure-resource-manager-templates"></a>Déployer des infrastructures d’application populaires sur Linux à l’aide de modèles Azure Resource Manager

Les charges de travail nécessitent généralement de nombreuses ressources pour fonctionner conformément à la conception. Les modèles Azure Resource Manager vous permettent de définir non seulement comment les applications sont configurées, mais également comment les ressources sont déployées pour prendre en charge les applications configurées. Cet article vous présente les modèles les plus populaires de la galerie et fournit des informations sur l’utilisation du portail Azure, de l’interface de ligne de commande Azure ou de PowerShell pour les déployer.
