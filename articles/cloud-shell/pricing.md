# <a name="pricing"></a>Tarification

## <a name="compute-cost"></a>Coût de calcul
Interpréteur de commandes Azure Cloud s’exécute sur un ordinateur fourni gratuitement, mais nécessite une toouse de partage de fichier d’Azure attaché. 

## <a name="storage-cost"></a>Coût de stockage
Interpréteur de commandes de cloud crée une image de 5 Go par défaut votre répertoire $HOME votre toopersist de partage de fichier. Les partages de fichiers Azure font l’objet de frais standard.

Cliquez [ici pour plus d’informations sur les coûts d’Azure](https://azure.microsoft.com/pricing/details/storage/files/).
