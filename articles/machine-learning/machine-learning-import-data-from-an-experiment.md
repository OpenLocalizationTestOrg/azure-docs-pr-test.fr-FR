# <a name="import-your-data-into-azure-machine-learning-studio-from-another-experiment"></a>Importer vos données dans Azure Machine Learning Studio à partir d’une autre expérience
[!INCLUDE [import-data-into-aml-studio-selector](../../includes/machine-learning-import-data-into-aml-studio.md)]

Il peut arriver lorsque vous souhaitez tootake un résultat intermédiaire à partir d’une expérience et utilisez-le comme partie d’une autre expérience. toodo, vous enregistrez le module de hello comme un jeu de données :

1. Cliquez sur sortie hello du module hello que vous souhaitez toosave comme un jeu de données.
2. Cliquez sur **Enregistrer comme jeu de données**.
3. Lorsque vous y êtes invité, entrez un nom et une description qui permettrait dataset hello tooidentify facilement.
4. Cliquez sur hello **OK** coche.

Lorsque hello enregistrez se termine, hello dataset sera disponible pour une utilisation dans des expériences dans votre espace de travail. Vous pouvez le trouver dans hello **jeux de données enregistrés** liste dans la palette de module hello.
