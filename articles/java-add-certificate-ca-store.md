    <span data-ttu-id="781d9-112">Vous êtes invité à entrer le mot de passe du magasin.</span><span class="sxs-lookup"><span data-stu-id="781d9-112">You'll be prompted for the store password.</span></span> <span data-ttu-id="781d9-113">Le mot de passe par défaut est **changeit**.</span><span class="sxs-lookup"><span data-stu-id="781d9-113">The default password is **changeit**.</span></span> <span data-ttu-id="781d9-114">(Pour le modifier, consultez la documentation relative à keytool à la page <http://docs.oracle.com/javase/7/docs/technotes/tools/windows/keytool.html>.)
