# <a name="azure-app-service-security"></a>Sécurité dans Azure App Service
Azure App Service respecte les principaux standards du secteur en termes de sécurité et de fiabilité pour fournir à nos clients une plateforme sécurisée. Nous disposons de plusieurs fonctionnalités simples d’utilisation pour aider à sécuriser le trafic entrant et sortant de votre application. Nous aidons également nos clients à sécuriser leur code d’application en fournissant des fonctionnalités en externe pour analyser votre application web et y rechercher des vulnérabilités.

[!INCLUDE [app-service-blueprint-security](../../includes/app-service-blueprint-security.md)]

Azure App Service utilise la même solution anti-programme malveillant qu’Azure Cloud Services et que les ordinateurs virtuels. Pour en savoir plus à ce sujet, consultez notre [documentation sur les logiciels anti-programme malveillant](../security/azure-security-antimalware.md).
