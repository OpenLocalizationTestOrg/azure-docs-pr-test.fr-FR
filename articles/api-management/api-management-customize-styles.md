> <span data-ttu-id="b63bf-125">Si vous connaissez [l'amorçage][bootstrap], ces règles sont en fait des [variables LESS][LESS variables] dans le thème d'amorçage utilisé par le portail des développeurs.</span><span class="sxs-lookup"><span data-stu-id="b63bf-125">If you're familiar with [bootstrap][bootstrap], these rules are in fact [LESS variables][LESS variables] within the bootstrap theme used by the developer portal.</span></span>
