
* [Inscrivez un ID d’application pour votre application](https://developer.apple.com/library/ios/documentation/IDEs/Conceptual/AppDistributionGuide/MaintainingProfiles/MaintainingProfiles.html#//apple_ref/doc/uid/TP40012582-CH30-SW991). Créer un ID d’application explicite (pas un ID d’application générique) et, pour **ID d’offre groupée**, utiliser les ID d’offre groupée exacte hello dans votre projet de démarrage rapide de Xcode. Il est également essentiel que vous sélectionnez hello **des Notifications Push** option. 
* Ensuite, [tooprepare pour la configuration des notifications push](https://developer.apple.com/library/ios/documentation/IDEs/Conceptual/AppDistributionGuide/AddingCapabilities/AddingCapabilities.html#//apple_ref/doc/uid/TP40012582-CH26-SW6), créer des certificats SSL un « Développement » ou « Distribution ».
