

Pour exécuter cet exemple, assurez-vous que vous avez installé la dernière version d’[Azure CLI 2.0](https://docs.microsoft.com/cli/azure/install-azure-cli). Pour démarrer, exécutez `az login` pour créer une connexion avec Azure.

Cet exemple fonctionne dans une interface d’interpréteur de commandes Bash. Pour en savoir plus les options d’exécution de scripts Azure CLI dans le client Windows, consultez la page [Running the Azure CLI in Windows (Exécution d’Azure CLI dans Windows)](../articles/virtual-machines/windows/cli-options.md).
