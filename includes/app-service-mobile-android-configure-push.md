
1. Dans le [Portail Azure](https://portal.azure.com/), cliquez sur **Parcourir tout** > **App Services**, puis cliquez sur votre serveur principal Mobile Apps. Sous **Paramètres**, cliquez sur **Push**, puis sur **Configurer les services de notification Push**.
2. Accédez à **Google (GCM)**, entrez la clé serveur hérité FCM obtenue à partir de la console Firebase, puis cliquez sur **Enregistrer**.

Votre service est désormais configuré pour fonctionner avec Firebase Cloud Messaging.
